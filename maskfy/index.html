Maskfy - Input Mask Simple

A Javascript library without a dependency of jQuery, Zepto, and etc ... Very simple to install and use. With only 1kb (gzip) code, it's also well accepted on mobile devices

Install / Usage

Add Maskfy file

<script src="path/js/maskfy.min.js"></script>

HTML (by class or attribute)

Insert a selector in class tag

<input class="page--input" data-mask="999.999.999-99" type="text" placeholder="CPF (Ex.: 999.999.999-99)"> or <input class="page--input mask-by-class" type="text" placeholder="CPF (Ex.: 999.999.999-99)">

Javascript Simple (by attribute)

<script> new Maskfy('[data-mask]'); </script> or <script> new Maskfy('.mask-by-class'); </script>

Javascript Options

<script> new Maskfy({ tag: '[data-mask]', // selector tag reverse: true, // reverse typing mask: '999.999.999,99', // mask pattern size: 3 // minimum digits }); </script>

Exemples

Data attributes [data-*]

If necessary, you could add attributes in the tag, to configure your mask options

data-mask

<input data-mask="999.999.999,99">

data-mask-reverse

<input data-mask="999.999.999,99" data-mask-reverse>

data-mask-size

<input data-mask="999.999.999,99" data-mask-reverse="" data-mask-size>
